US25. ModificarNom

Modificar el nom del Perfil de l'usuari. S'utilitzen 3 tests d'acceptació. Com a Usuari registrat, vull modificar el nom del meu perfil, perquè aixì puc actualitzar les dades del meu Perfil.

Test Acceptació 25.1: Modificació nom correct

En cas que l'Usuari vulgui modificar el seu nom de perfil, quan introdueixi el nou nom, el Sistema substituirà amb el nom nou l'antic nom.

Nom Client Nom Usuari Antic Nom Usuari Nou Sucess
ajaleo Manuel Alberto Se ha cambiado el nombre correctamente a: Alberto
dtomacal Marco Julian Se ha cambiado el nombre correctamente a: Julian
chachipistachi Ignasi Carlos Se ha cambiado el nombre correctamente a: Carlos
ajaleo Pol Ruben Se ha cambiado el nombre correctamente a: Ruben

Test Acceptació 25.2: Error modificació nom

En cas que l'Usuari vulgui modificar el seu nom de perfil, però aquest nom ja existeixi dins del client, quan introdueixi el nou nom, el Sistema mostrará un missatge d'error indicant que aquest nom ja existeix i no modificará res.

Nom Client Nom Usuari Antic Nom Usuari Nou Sucess
ajaleo Manuel Pol User alredy has this name
dtomacal Marco Ignasi User alredy has this name
chachipistachi Ignasi Ignasi User alredy has this name

Test Acceptació 25.3: Usuari no introdueix nom

En cas que l'Usuari no vulgui modificar cap nom, quan no introdueixi un nou nom, el Sistema no fará res i el nom no es modificarà.

Nom Client Nom Usuari Antic Nom Usuari Nou Sucess
ajaleo Manuel El nombre del usuario nuevo no debe ser vacío
dtomacal Marco El nombre del usuario nuevo no debe ser vacío
chachipistachi Ignasi El nombre del usuario nuevo no debe ser vacío
