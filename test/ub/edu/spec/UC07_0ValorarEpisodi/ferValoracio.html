US15. FerValoració

Permet a l'usuari escollir quin tipus de valoració voll fer. S'utilitzen 3 tests d'acceptació. Com a Usuari registrat, vull valorar un episodi, perquè així l'app pugui recomanar-me noves sèries semblants.

Test Acceptació 15.1: Valorar amb Cor

En cas que l'Usuari demani valorar amb Cor, quan seleccioni l'opció de valorar amb Cor, el Sistema li mostrarà la pestanya per a poder valorar en aquest format (amb Cor)

Test Acceptació 15.2: Valorar amb Estrellas

En cas que l'Usuari demani valorar amb Estrelles, quan seleccioni l'opció de valorar amb Estrelles, el Sistema li mostrarà la pestanya per a poder valorar en aquest format (amb Estrellas)

Test Acceptació 15.3: No valorar

En cas que l'Usuari no vulgui fer cap valoració, quan seleccioni l'opció cancel·lar, el Sistema no modificará res i tancará la pestanya de Valorar
